# Cinema booking system
## 这是一个模拟电影院订票系统的小demo。
### * 共分3大部分：
##### 1.首页：电影资讯和热门推荐的展示。
##### 2 影院： 影院地区分布以及各门店的展开和合拢。
##### 3 订座：
######            * 3.1 可通过来回滑动电影海报，来点击选择电影。
######            * 3.2 可选择具体的观影场次，并进入相应的选座页面。
######            * 3.3 进入订票结算页面。
